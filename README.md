# Hash-Driven-Dynamic-Encoding-for-Secure-and-Biologically-Inert-DNA-Data-Storage

Repository Scope & Theoretical Boundary
This repository implements the core Source Coding Module (State-Dependent Dynamic Mapping, SDDM) proposed in the manuscript. It rigorously demonstrates the deterministic generation of biochemically inert DNA sequences (GC constraint, homopolymer suppression, ORF disruption) driven by cryptographic hash functions.
